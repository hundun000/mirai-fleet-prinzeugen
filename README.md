# ZacaFleetBot

## 简介

插件功能：使bot扮演角色欧根，作为舰队collection游戏助手。

- 推送微博： 艦colle鎮守府情報
- 事项提醒： 整点报时
- 舰娘百科查询： 舰娘信息（舰娘百科网页链接、改造链、初始装备……）
- 闲聊

依赖服务：mirai框架、微博api、舰娘百科api
